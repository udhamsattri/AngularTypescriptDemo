# AngularTypescriptDemo

Sample application using Angular 1.x with Typescript, npm, Karma and Jasmine with VS2015 as code editor.


## Features
1. Angular module. 
2. Directive.
3. Domain
4. Interface. 
5. Service. 
6. Create routes. 
7. Use Promises. 
8. Http GET method 
9. Http POST method. 
10. Unit Test
